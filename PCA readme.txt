This zip file includes the datasets, codes and output required to run the project. The below explined points are important for clarification of file.

1) Food is the actual dataset https://www.efsa.europa.eu/en/data-report/food-consumption-data
2) Foods_cleaned_data is the cleaned dataset got from the PCA_clean code.
3) pcaspark1 is the pyspark code which ran on cluster with the help of Foods_cleaned_data.
4) Output of pcaspark1 code is stored as PCA_expvariance and PCA_Output file.
5) Both csv files are useful to make visualization on local python's Jupyter Notebook.
